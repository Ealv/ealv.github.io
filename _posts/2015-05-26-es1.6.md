# Let et Const 


## "Problème" : Hoisting

```js
function f(){
	console.log(a);
}
```

Ce code renvoie : 
Uncaught 
ReferenceError: a is not defined

```js
function f(){
	console.log(a);
	var a;
}
```
Ce code renvoie : 
undefined

Pourquoi ? les déclarartions de variables aveec le mot clé var sont déplacer en début de leurs scopes lors du pré-processing (hoisting).

```js
function f(b){
	if(true) {
		var a;
		a = 10;
	}
	console.log(a);//return 10, a is hoisting
}
```

##Solution/Nouveauté Javascript 1.6 : Let !


```js
function f(b){
	if(true) {
		let a;
		a = 10;
	}
	console.log(a);//ReferenceError: a is not defined
}
```

##Nouveauté  : Const !

function f(b){
